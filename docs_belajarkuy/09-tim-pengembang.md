# 09 — Tim Pengembang

BelajarKUY dikembangkan oleh tim mahasiswa sebagai Tugas Besar mata kuliah Pemrograman Web. Pembagian tanggung jawab berikut disusun berdasarkan kontribusi nyata pada riwayat pengembangan proyek.

## 9.2 Catatan

Pembagian peran di atas bersifat fokus utama. Dalam praktiknya, anggota tim saling membantu lintas bidang sesuai kebutuhan sprint. Rincian kontribusi setiap anggota dapat ditelusuri melalui riwayat komit repositori.
